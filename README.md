# ✅ Fix Auto-Import Not Working in VS Code (Quick Solution)

If VS Code is not showing auto-import suggestions after installing a new package, use this simple fix:

---

## ✔ Quick Fix

1. Press **Ctrl + Shift + P**  
2. Type: **"TypeScript: Restart TS Server"**  
3. Hit **Enter**

---

This instantly refreshes VS Code’s TypeScript engine and restores auto-import suggestions.
